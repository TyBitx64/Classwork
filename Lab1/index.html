AMD Ryzen

The latest AMD desktop APU's of the 8000 & 9000 series allow the users to be able to run their favourite games without the need to pair it with a dedicated graphics card.

This is achieved is through a process that AMD calls Infinity Fabric™ and paired with their FSR upscaling graphics, gamers can play games on low graphic settings that look like high graphic settings.
